Status Codes

Status Code Meaning Additional Comments
1 Success Sent this status code after successful Login (along with respective Authenticated Object), or successful updation and deletion
-4 User Already Registered Sent when trying to register Committees, Students or Admins
-1 Wrong Password Sent when trying to login
-2 User Not Verified. Wait for Verification Sent when trying to login
-3 User Not Registered Sent when trying to login
0 Something Went Wrong Sent when trying to login, register, update or delete
